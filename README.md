# 🛡️ PhishGuard Simulator — Cybersecurity Awareness Tool

PhishGuard Simulator is an ethical cybersecurity training project designed to demonstrate how phishing attacks work in a controlled and safe environment. The system is built for education, helping users recognize fake login pages, suspicious links, and social engineering techniques.

---

# 🚀 Features

- Simulated phishing page detection training
- Fake login page awareness demos (non-functional credential capture)
- URL analysis and warning system
- Educational breakdown of phishing techniques
- Safe offline/local environment usage
- Cybersecurity awareness modules

---

# 🎯 Purpose

This project is built strictly for:

- Cybersecurity education
- Ethical hacking practice
- Phishing awareness training
- Security research in controlled environments
- Learning how attackers design fake login pages

---

# ⚠️ Disclaimer

This project is strictly for educational and ethical use only.

It does NOT support real-world phishing, credential theft, or targeting of any individuals, platforms, or services.

Unauthorized use of phishing techniques is illegal and punishable under cybersecurity laws.
